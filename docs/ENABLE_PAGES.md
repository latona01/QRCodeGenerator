# 🚀 GitHub Pages 有効化手順

## ✅ ファイルのプッシュが完了しました！

リポジトリURL: https://github.com/latona01/QRCodeGenerator

## 📝 次のステップ: GitHub Pagesを有効化

### 1. リポジトリのSettingsを開く
以下のURLにアクセス：
```
https://github.com/latona01/QRCodeGenerator/settings/pages
```

または：
1. リポジトリのページで「**Settings**」タブをクリック
2. 左メニューから「**Pages**」を選択

### 2. GitHub Pagesの設定

以下の設定を行います：

- **Source**: `Deploy from a branch` を選択
- **Branch**: `main` を選択
- **Folder**: `/ (root)` を選択
- 「**Save**」をクリック

### 3. デプロイを待つ

設定後、**5-10分**待ってからアクセスしてください。

### 4. 公開URL

以下のURLでアクセス可能になります：
```
https://latona01.github.io/QRCodeGenerator/
```

**注意**: リポジトリ名に大文字が含まれる場合、URLは小文字になります：
```
https://latona01.github.io/qrcodegenerator/
```

## 🔍 デプロイ状況の確認

リポジトリの「**Actions**」タブでデプロイの進行状況を確認できます。

## ✅ 完了後の確認

1. 上記のURLにアクセス
2. QRコード生成ツールが表示されることを確認
