# 🔲 QRコード生成ツール

シンプルで実用的なQRコード生成ツールです。**Web版**と**デスクトップGUI版**の2つのバージョンがあります。

## ✨ 機能

- **簡単操作**: テキストを入力してボタンをクリックするだけ
- **多様な入力形式に対応**: URL、テキスト、メールアドレス、Wi-Fi情報など
- **カスタマイズ可能**: サイズと誤り訂正レベルを調整可能
- **画像ダウンロード**: 生成したQRコードをPNG画像として保存
- **レスポンシブデザイン**: PC・スマホ・タブレットで使用可能（Web版）
- **モバイル最適化**: タッチ操作に最適化されたUI（Web版）
- **デスクトップGUI**: Python Tkinterベースのスタンドアロンアプリ（GUI版）
- **オフライン対応**: 基本的な機能はオフラインでも動作

## 🚀 使い方

### 📱 Web版（ブラウザ）

**推奨**: スマートフォンやタブレットでも使用可能。インストール不要。

1. **Windows (バッチファイル)**:
   ```bash
   start.bat
   ```

2. **Windows (PowerShell)**:
   ```powershell
   .\start.ps1
   ```

3. **手動起動**:
   - `index.html` をブラウザで開く

4. **モバイル端末**:
   - `index.html` をスマートフォンやタブレットのブラウザで開く
   - ホーム画面に追加してアプリのように使用可能（iOS/Android）

### 🖥️ GUI版（デスクトップアプリ）

**推奨**: デスクトップで頻繁に使用する場合。オフラインで完全に動作。

- Python 3.7以上

#### 起動方法

1. **Windows (バッチファイル)**:
   ```bash
   start_gui.bat
   ```

2. **Windows (PowerShell)**:
   ```powershell
   .\start_gui.ps1
   ```

3. **直接実行**:
   ```bash
   python qr_generator_gui.py
   ```

#### GUI版の特徴

### 基本的な使い方

1. テキストエリアにQRコード化したい内容を入力
   - URL: `https://example.com`
   - テキスト: `こんにちは、QRコード！`
   - メール: `mailto:example@email.com`
   - Wi-Fi情報: `WIFI:T:WPA;S:ネットワーク名;P:パスワード;;`

2. オプションを調整（任意）
   - **サイズ**: QRコードの大きさ（100-1000px）
   - **誤り訂正レベル**: 汚れや破損に対する耐性（詳しくは後述）
     - **L (約7%)**: 小さく、綺麗な環境向け（印刷物、ディスプレイなど）
     - **M (約15%)**: 標準、推奨（一般的な用途に最適）
     - **Q (約25%)**: 多少汚れやすい環境向け（屋外看板など）
     - **H (約30%)**: 最高耐性（屋外、劣化しやすい環境向け）

3. 「QRコードを生成」ボタンをクリック

4. 生成されたQRコードを確認し、必要に応じて「PNG画像としてダウンロード」をクリック

## 📖 誤り訂正レベルについて（詳しく）

誤り訂正レベルは、QRコードがどれくらいの汚れや欠損に耐えられるかを表します。

### 実際の選び方

- **L (Low - 約7%)**: 最も小さいQRコード。綺麗な環境（名刺、印刷物、ディスプレイなど）向け。汚れや破損に弱い。
- **M (Medium - 約15%)**: **標準設定（推奨）**。バランスが良く、一般的な用途に最適。多少の汚れには強い。
- **Q (Quartile - 約25%)**: 多少汚れやすい環境向け。屋外看板、工場など。QRコードがやや大きくなる。
- **H (High - 約30%)**: 最も高耐性。劣化しやすい環境（屋外、長時間使用など）向け。QRコードが最も大きくなる。

### 実際の使用例

- **名刺やチラシ（綺麗な環境）**: L または M
- **ディスプレイ表示**: M（標準）
- **屋外看板**: Q または H
- **工場や倉庫（汚れやすい）**: H

一般的には **M（Medium）** を選択すればほとんどの用途に対応できます。
